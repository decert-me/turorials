# Decert.me 教程

[Decert.me](https://decert.me/) 是一个开源、开放的公益平台，鼓励大家通过 Decert.me 在学习的同时构建自己的链上信用履历。 Decert.me 相信只有开放、可信的数据才能促进形成一个更自由、更可信、更高效的世界。

本项目包含多个教程文档，目前已有 区块链基础 和 学习 Solidity 等。

所有教程见：https://decert.me/tutorials
